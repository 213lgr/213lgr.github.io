1、认识简谱构造、认识唱名

简单介绍简谱中的几个成分

介绍唱名：

介绍音名： 1 = C （表示开始的键位）（结合音名和唱名才能知道K的位置）

音名从钢琴来了解音名很好理解

介绍：钢琴有四组，每组键数量一样，一组7个白键，每组都以同样的方式来命名按键。
键盘命名规则：白键都写在上面了就是字母，

这里是提取了其中两组来展示，所以有两组C到B。

当1 = C 时候 do 就对应C键开始，依次类推 D = 2 re 对应D ，E = 3 对应 mi
1也可以等于其他的：1 = D ，1 = E ；在音效的区别为K（k后面会提到）不一样，音调要高一点点

问题扩展：

钢琴键位的问题：
1 = C

1后面的数字（2,3,4,5,6,7）都在白键上
1= D

但是有几个数字跑到黑键上去了。
1 = E

也有几个在黑键上。
其实只有1 = C才都在白键上。
原因：
1、键盘黑键命名

黑键是相对白键的命名，可以是上个白键升半调，也可以是下个白键降半调。
所以黑键有两种命名。

2、半音：相邻两个按键的距离，C和C#的距离是一个半音，D#和E的距离是一个半音。
相邻：指的是白键和黑键一起算，（白和黑地位相当，看起来不同是是钢琴构造的优化）
所以：C和D就不相邻

3、全音：两个半音的距离，C到D是一个全音，D#到F（的距离）是一个全音

4、1-7的内在规则(和钢琴无关）：do 到 se 的距离不相等

3到4音高变化小一点，7到下一级的1的音高变化也比较小
如果其他的音高变化是1,那3-4 7-i 的变化就是0.5
距离就是 ：1、 1 、0.5 、 1 、1 、1 、0.5
也就是：全、全、半、全、全、全、半

5、对应关系
钢琴黑白键平等

可以看成这样的
1 = C

1 = D

距离关系不变，开始键变了，有些数字就到黑键上去了


把黑键变小提上去，就变成这样了

3、（自然）大调

自然大调最常用，为了表达方便，就把“自然”隐去了，简称大调
全全半全全全半就是（自然）大调的规则
1 = C 称为 C大调
1 = D 称为 D大调

3、音阶

声音像阶梯一样
按调式构建出来的声音按键

所以D大调的音阶就是 D 、E 、F# 、G 、A 、B 、C#、 D
当然 F#也可以叫Gb 但是这就会不按字母顺序排列，为了方便就叫F# ，同理 用C#名字，不用Db名字
